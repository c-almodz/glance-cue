# Glance Cue

**Read naturally. Present confidently.**

Glance Cue is a lightweight Windows overlay notes and cueing app designed for meetings, interviews, presentations, demos, and live speaking.

It helps you keep your notes visible on screen while staying focused on your audience or camera.

---

## Free Public Beta

Glance Cue is currently available as a **free public beta**.

This release is intended for early testing and feedback. Some bugs may still exist, and features may change in future versions.

---

## Features

* Overlay notes for meetings and presentations
* Smooth scrolling cue/teleprompter mode
* Bookmarks for quick navigation
* Reading guide and marker tools
* On-top presentation mode
* Basic typography and layout controls
* Windows installer and portable version

---

## Download

You can download the latest beta release from the **Releases** section:

* `GlanceSetup-Beta.exe` — recommended installer
* `Glance-Portable-Beta.zip` — portable version

* Windows 10
* Windows 11
